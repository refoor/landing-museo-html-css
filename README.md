# Landing Page de Museo

Página web con temática de museo desarrollada como práctica de HTML y CSS.

## Descripción

Este proyecto consiste en una landing page para un museo, con una sección principal visual, menú de navegación y diseño enfocado en la presentación de una institución cultural.

## Tecnologías utilizadas

- HTML
- CSS

## Objetivo del proyecto

Practicar la estructura de una página web, el uso de imágenes de fondo, la creación de menús de navegación y la organización visual de una landing page.

## Características

- Menú de navegación.
- Imagen principal de presentación.
- Texto central destacado.
- Diseño visual con temática cultural.
- Estructura básica de sitio web informativo.

## Nota

Este proyecto fue realizado como práctica inicial, por lo que está optimizado principalmente para una resolución de escritorio y puede no contar con diseño responsive completo.

## Mejoras futuras

- Adaptar el diseño a celulares y tablets.
- Agregar más secciones informativas.
- Mejorar la responsividad.
- Optimizar imágenes.
- Agregar interactividad con JavaScript.
